# Анализ банковских транзакций с использованием Hadoop и PySpark

## Описание задачи
В данной работе реализован процесс обработки и анализа логов банковских транзакций. Основная цель — выявить паттерны активности клиентов, отфильтровать неудачные операции и визуализировать нагрузку на банковскую систему в зависимости от времени суток и дня недели.

**Вариант:** Бизнес-задача "Банковские транзакции".

## Инструкция по запуску
Для запуска решения необходимо наличие среды Docker.

1.  **Запуск контейнера:**
    ```bash
    docker run -it -p 8888:8888 jupyter/pyspark-notebook:latest /bin/bash
    ```
2.  **Загрузка данных:** Поместите файл `transaction_data.csv` в рабочую директорию контейнера.
3.  **Установка библиотек:**
    ```bash
    pip install pyspark pandas seaborn matplotlib
    ```
4.  **Запуск анализа:**
    ```bash
    python lab_01.py
    ```

## Источник данных
Данные сгенерированы синтетически на основе типовых логов банковских систем (включают ID транзакции, сумму, статус, временную метку и тип устройства).
